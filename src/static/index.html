API Endpoints

Basic Usage

GET /avatar/:name

With Auto Initials

GET /avatar/:name?initials=auto

With Custom Initials

GET /avatar/:name?initials=OA

With Custom Color

GET /avatar/:name?color=black

With Custom Width

GET /avatar/:name?w=120

SVG Response

GET /avatar/:name?type=svg

GitHub Username

GET /avatar/github/:username
Response: image/svg+xml or image/png
